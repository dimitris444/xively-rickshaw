Xively-rickshaw

Some example time series graphs generated from Xively using Rickshaw

About

Xively (formerly Cosm, formerly Patchube) has a Javascript library for their API. Here are some examples using the Rickshaw library to render timeseries data for various devices. I also mixed in the moment.js library to do some date munging.

Examples

  • Current cost electricity monitor
  • Air Quality Egg
  • Brixton Energy single day
  • Brixton Energy several weeks
